예매안내

  • 예매일시

    - 1차 오픈 4월 03일(수), 오후 2시
    * 대상일 : 2024. 4. 17. - 5. 20.
    - 2차 오픈 4월 30일(화) 오후 2시
    * 대상일 : 2024. 5. 22-6.24.

  • 예매금액

    15,000원 (궁중병과 세트 1종 + 궁중약차 1종)
    * 1인 2매 예매 가능

  • 할인정보

    장애인 1-3급(동반 1인까지) 및 4급 이하(본인만)50% 할인, 국가유공자(본인만)50% 할인
    * 현장에서 관련 증빙자료(복지카드 등) 확인 후 입장 가능

예매방법

  • 온라인 사전 예약

    ① 예매 사이트(티켓링크) 접속 및 로그인
    ② 참여 일자, 회차, 다과 세트, 좌석 선택 후 비용(카드) 결제
    *(예매 시) 대추인절미병 세트 / 주악 세트 중 선택
    *궁중약차는 현장에서 선택 후 수령
    ③ 생과방 입구(안내처)에서 본인 확인▶ 약차 선택 후 차패 수령

  • 전화예매

    - 예매대상 : 만 65세 이상, 장애인, 국가유공자
    - 예약시간 : 평일 10:00~19:00, 주말·공휴일 휴무
    * 현장에서 증빙자료를 확인 할 예정이오니, 신분증(경로우대증, 국가유공자증, 장애인복지카드 등)을 반드시 지참하여 주시기 바랍니다.
